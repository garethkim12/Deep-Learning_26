# PyTorch nn.Linear 및 nn.BCELoss 실습
본 실습은 이전의 수동 파라미터 업데이트 방식에서 벗어나, PyTorch의 신경망 구축을 위한 핵심 모듈인 torch.nn.Linear와 torch.nn.BCELoss를 사용하여 모델 학습 구조를 더 체계적이고 표준화된 방식으로 구현하는 방법을 다룹니다.

## 🛠 주요 학습 내용
 1. nn.Linear(1, 1) 도입: 가중치(weight)와 편향(bias)을 직접 생성하는 대신, PyTorch에서 제공하는 Linear 레이어를 사용하여 선형 모델($H(x) = a \cdot X + b$)을 구축합니다.
 2. nn.BCELoss: 직접 손실 함수 수식을 작성하지 않고, BCELoss 모듈을 사용하여 이진 교차 엔트로피 비용을 효율적으로 계산합니다.
 3. 데이터 Shape 최적화: nn.Linear 레이어에 데이터를 전달하기 위해 데이터를 (n, 1) 형태의 2차원 텐서로 변환하는 과정을 익힙니다.
 4. 표준 학습 루프 구현: linear.parameters()를 옵티마이저에 전달하여 학습 대상을 자동으로 관리하고, 모델 파라미터의 기울기를 자동으로 추적하는 파이토치식 학습을 구현합니다.
 
 ## 🚀 학습 순서
  1. 데이터 전처리: 넘파이 배열을 활용해 데이터를 준비하고, reshape(-1, 1)을 통해 텐서의 차원을 모델 입력 형태에 맞게 조정합니다.
     모델 및 손실 함수 구성: torch.nn.Linear로 모델을, torch.nn.BCELoss로 비용 평가 기준을 설정합니다.
  
  2. 랜덤 시드 고정: torch.manual_seed(42)를 사용하여 모델 초기값이 동일하게 유지되도록 설정합니다.
  
  3. 최적화 설정: linear.parameters()를 torch.optim.SGD에 전달하여 파라미터 자동 업데이트를 준비합니다.
  
  4. 학습 루프 실행: zero_grad(), forward, backward, step으로 이어지는 표준 학습 과정을 수행합니다.예측 단계: 학습된 linear 모델에 새로운 키 데이터를 넣어 농구선수 확률을 추론합니다.
  
  
## 💡 이전 실습과의 차이점
   ▶ 파라미터 관리: a, b 변수를 직접 선언하지 않고 linear.weight, linear.bias 속성을 통해 접근 및 관리합니다.
   
   ▶ 입력 차원: 레이어 처리 방식에 따라 입력 텐서의 형태를 (데이터 개수, 특성 개수)인 (4, 1)로 유지합니다.
   
   ▶ 기울기 확인: a.grad 대신 linear.weight.grad를 사용하여 모델의 미분값을 확인합니다.
