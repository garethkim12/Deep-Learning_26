경사 하강법 (Gradient Descent) 기초 학습
이 프로젝트는 파이썬과 판다스(pandas)를 사용하여 선형 회귀의 핵심 알고리즘인 경사 하강법을 이해하고 구현하는 과정을 담고 있습니다.

목차
데이터 준비

예측 모델 설정 및 오차 계산

Cost(비용) 함수 이해

최적의 파라미터 찾기

1. 데이터 준비
판다스(pandas) 라이브러리를 사용하여 기온(temperature)에 따른 떡볶이 판매량(sales) 데이터를 정의하고 데이터프레임으로 시각화합니다.

입력값(X): 기온

정답(Y): 판매량

2. 예측 모델 설정 및 오차 계산
선형 모델 예측값 = a * X + b에서 기울기(a)와 절편(b)을 임의로 설정한 뒤, 각 데이터 포인트에 대한 예측값과 오차를 계산합니다.

오차: 예측값 - 실제값

3. Cost(비용) 함수 이해
모델의 성능을 평가하기 위해 Cost 함수를 정의합니다.

방식: 오차를 제곱한 값들의 평균(Mean Squared Error)을 구하여 모델이 데이터를 얼마나 잘 설명하는지 수치화합니다.

함수: calculate_cost(a, b)를 통해 특정 a, b값에 대한 Cost를 계산하여 반환합니다.

4. 최적의 파라미터 찾기
b를 고정한 상태에서 다양한 a(기울기) 값을 변화시키며 각 경우의 Cost를 계산합니다. Cost가 가장 작은 지점을 찾아가는 것이 경사 하강법의 핵심 과정임을 실습합니다.
