### 1. 로그(Log) 함수의 이해
딥러닝 모델의 성능을 평가할 때 사용하는 로그 함수의 개념을 익힙니다.

정의: log(x, base) 함수는 밑(base)을 몇 번 곱해야 x가 되는지를 계산합니다.

중요성: 이진 분류에서는 실제값과 예측값의 차이를 계산할 때 로그 함수를 사용하며, 특히 확률값(0~1)을 다룰 때 매우 유용합니다.

### 2. 이진 분류에서의 비용 함수 (Binary Cross-Entropy)
이진 분류 모델의 오차를 정의하는 비용 함수(Cost Function)를 구현합니다.

수식: Cost = - (y * log(z) + (1 - y) * log(1 - z))

y: 실제 정답 (0 또는 1)

z: 모델의 예측 확률값 (시그모이드 함수의 결과)

작동 원리: 정답이 1일 때는 log(z)가, 정답이 0일 때는 log(1-z)가 작동하여 오차가 커질수록 Cost가 높아지도록 설계되었습니다.

### 3. 평균 비용(Mean Cost) 계산
모델 전체의 성능을 하나의 지표로 확인하기 위해 각 데이터 샘플의 비용을 평균 냅니다.

NumPy 활용: 모델에 들어온 여러 데이터의 비용을 계산한 뒤, np.mean()을 사용하여 전체 평균 Cost를 구합니다.

이 값이 작아질수록 모델이 데이터를 잘 분류하고 있음을 의미합니다.
