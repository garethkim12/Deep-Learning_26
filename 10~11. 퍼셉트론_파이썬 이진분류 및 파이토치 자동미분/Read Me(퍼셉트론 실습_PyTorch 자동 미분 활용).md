# 퍼셉트론 실습: PyTorch 자동 미분(Autograd) 활용
이 프로젝트는 수동으로 기울기를 계산해야 했던 이전 퍼셉트론 구현 방식에서 벗어나, PyTorch의 자동 미분(Autograd) 기능을 사용하여 모델의 파라미터를 최적화하는 과정을 실습합니다.

## 목차
-.실습 환경 및 라이브러리

-.데이터 준비 및 정규화

-.PyTorch Autograd 개념

-.모델 학습 루프 구현

-.결과 예측

1. 실습 환경 및 라이브러리
본 실습은 수치 계산을 위한 numpy와 딥러닝 프레임워크인 torch를 함께 사용합니다.

torch: 모델 연산 및 자동 미분(autograd) 기능을 담당합니다.

2. 데이터 준비 및 정규화
이전과 동일하게 사람의 키(X)와 농구선수 여부(y) 데이터를 사용합니다.

모델이 효율적으로 학습할 수 있도록 numpy를 사용하여 입력 데이터를 평균과 표준편차로 정규화하는 전처리 과정을 포함합니다.

3. PyTorch Autograd 개념
이 실습의 핵심입니다. 사람이 직접 미분 공식(편미분)을 유도하여 코딩할 필요 없이, PyTorch가 계산 그래프를 따라가며 자동으로 기울기를 계산해 줍니다.

requires_grad=True: 가중치(a)와 편향(b) 텐서에 이 옵션을 설정하면, PyTorch는 해당 변수에 대한 미분값을 추적합니다.

backward(): Cost 함수의 최종 결과값에서 이 메서드를 호출하면, 전체 연산 과정에 대한 기울기가 자동으로 계산되어 각 변수의 .grad 속성에 저장됩니다.

4. 모델 학습 루프 구현
반복문(Epoch)을 통해 경사 하강법을 실행합니다.

순전파(Forward): 입력값과 가중치를 연산하여 예측값을 구합니다.

비용 계산: 이진 분류를 위한 비용 함수를 정의합니다.

역전파(Backward): cost.backward()를 호출하여 자동으로 기울기를 얻습니다.

파라미터 업데이트: a -= learning_rate * a.grad와 같이 저장된 기울기를 사용하여 모델 파라미터를 업데이트합니다.

5. 결과 예측
학습이 완료된 후, 새로운 데이터를 입력받아 농구선수일 확률을 도출하고 분류 결과를 출력합니다.
