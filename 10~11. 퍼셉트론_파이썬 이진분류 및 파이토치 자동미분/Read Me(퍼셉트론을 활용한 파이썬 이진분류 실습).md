# 퍼셉트론을 활용한 파이썬 이진 분류 실습
이 프로젝트는 인공지능의 가장 기초적인 모델인 퍼셉트론(Perceptron)을 이해하고, 특정 데이터(사람의 키)를 기반으로 농구선수 여부를 예측하는 이진 분류 모델을 구축하는 과정을 담고 있습니다.

## 목차
-.데이터 준비 및 전처리

-.퍼셉트론 모델 구조

-.학습 프로세스

-.새로운 데이터 예측

1. 데이터 준비 및 전처리
모델 학습을 위해 입력값 X(키)와 정답 y(농구선수 여부: 0 또는 1) 데이터를 정의합니다.

정규화(Normalization): 학습의 효율성을 높이기 위해 입력값을 평균과 표준편차를 사용하여 정규화합니다. 이 과정은 모델이 더 빠르고 안정적으로 수렴하도록 돕습니다.

2. 퍼셉트론 모델 구조
퍼셉트론은 입력값에 가중치(weight, 기울기 a)를 곱하고 편향(bias, 절편 b)을 더한 뒤, 시그모이드(Sigmoid) 함수를 통과시켜 0과 1 사이의 확률값으로 변환하는 구조를 가집니다.

3. 학습 프로세스
경사 하강법(Gradient Descent)을 사용하여 비용(Cost) 함수를 최소화하는 방향으로 모델을 학습시킵니다.

반복 학습(Epoch): 정해진 횟수만큼 데이터를 반복 학습하며 파라미터 a와 b를 조금씩 수정합니다.

비용 계산: 예측값과 실제 정답 사이의 오차를 계산하여 모델을 점진적으로 개선합니다.

4. 새로운 데이터 예측
학습이 완료된 모델에 새로운 입력값(예: 키 185cm)을 넣어 농구선수일 확률을 계산합니다.

결과 판별: 산출된 확률값이 0.5 이상이면 "농구선수", 미만이면 "농구선수 아님"으로 분류합니다.
